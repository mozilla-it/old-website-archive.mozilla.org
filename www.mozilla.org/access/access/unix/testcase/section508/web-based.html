Web-based Intranet and Internet Information and Applications Checklist

Maintained by Jessie Li

Click the test case name to see the test case description:

Checklist
Description
Test Case Name
(a)Text Tags

 A text equivalent for every non-text element shall be provided.

 text_tag

(b)Multimedia Presentations

 Equivalent alternatives for any multimedia presentation shall be synchronized with the presentation.

 mul_presentation

(c)Color

 Web pages shall be designed so that all information  with conveyed color is also available without color.

 color

(d)Readability

 Documents shall be organized so they are readable without requiring an associated style sheet.

 readable

(e)Serve-Side Image Maps

 Redundant text links shall be provided for each active region of a server-side image map.

 		N/A

(f)Client-Side Image Maps

 Client-side image maps shall be provided instead of server-side image maps except where the regions cannot be defined with an available geometric shape.

 		cs_image

(g)Data Table

 Row and column headers shall be identified for data tables.

 		data_table_1 

(h)Data Table

 Markup shall be used to associate data cells and header cells for data tables that have two or more logical levels of row or column headers.

 data_table_2

(i)Frames

 Frames shall be titled with text that facilitates frame identification and navigation.

 frames

(j)Flicker Rate

 Pages shall be designed to avoid causing the screen to flicker with a frequency greater than 2 Hz and lower than 55 Hz. N/A
(k)Text-Only Alternative

 If accessibility cannot be accomplished in any other way, provide a text only page with equivalent information or functionality. Update the content of the text only page whenever the primary page changes.

 N/A
(l)Scripts

 When pages utilize scripting languages to display content, or create interface elements, the information provided by the script shall be identified with functional text that can be read by assistive technology. script

(m)Applets and Plug-Ins

 When an applet, plug-in or other application is required to be present, provide a link to one that is directly accessible, or provide alternate content for those which are not directly accessible.

 applet_plugin

(n)Electronic Forms

 The on-line form allow user using assistive technology to access the information to complete and submit the form.

 eletro_form

(o)Navigation Links

 A method shall be provided that permits users to skip repetitive navigation links.

 nav_link

(p)Time Delays

 When a timed response is required, the user shall be alerted and given sufficient time to indicate more time is required.

 N/A
