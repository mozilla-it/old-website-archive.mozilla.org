Firefox 1.5 Accessibility Status

Key goals: compatibility with Window-Eyes 5.5 and JAWS 7.0 screen readers, section 508 compliance under Windows 98 and up.

Maintained by Mozilla Accessibility Community.

Current builds: Nightly build for Firefox 1.5

Attempting to fix for Firefox 1.5

- no bug#: datagrid crash with Window-Eyes (waiting for minimal test case from Becky)

Issues that will slip

Bugzilla query for Critical and Major VPAT issues on Windows for Firefox 1.5
- bug 310241: JS text quoting function that no longer works

- bug 309027: save as for images not working in some circumstances related to download manager. Also fixes bug 312227. (bryner)
- bug 193316: caret not visible in readonly field (baking on trunk)

 - bug 304497: RSS not accessible, will fix for Firefox 1.5.1  [ VPAT issue ]
- bug 305797 setup screens font size issues  [ VPAT issue ]
- bug 308560: automatic immediate refreshes give appearance of broken page load for screen reader users.
- bug 310568: can we get "accessibility" listed as a problem in the reporter?
- bug 93149:  No way to move focus between plugin and browser from keyboard  [ VPAT issue ]
- bug 307741: Tp (page load performance) regression on creature from focus controller changes
