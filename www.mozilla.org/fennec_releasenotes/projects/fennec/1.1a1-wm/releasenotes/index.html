This is fourth alpha version version of Fennec for Windows Mobile. This release is provided for testing purposes only, intended to:



  • Get wider feedback on our approach to the user experience
  • Engage Mozilla community teams, including localizers, and testers
  • Get feedback from Web developers
  • Encourage add-on developers to port existing add-ons and invent new ones for mobile


Fennec is built on the same browser engine as the desktop Firefox 3.6 with some extensive under the hood work to optimize for mobile. These key design principles are at the heart of the browsing experience: minimize typing, let the Web have center stage, and personalize your browser, to name a few. Fennec 1.1 alpha 1 for Windows Mobile includes some of your favorite features including the Awesome Bar, Weave Sync, add-ons, and Location-Aware Browsing.



This release continues to focus on stability, performance, and integration with Windows Mobile. This version also includes numerous fixes, better performance and further refinement of our theme.


To help out, visit the Mozilla mobile page to learn about our public weekly meetings, how to find our community on IRC, and more.

These release notes provide a "quick start" guide, some highlights of what's included in this version, download and installation instructions, known issues, and frequently asked questions. Please read these notes and the bug filing instructions before reporting bugs to Bugzilla. As always, we welcome and encourage your feedback.


What's in this version?

New since 1.0 alpha 3:

  • Improved interactions with the software keyboard
  • Fixed painting artifacts from panning
  • Faster .exe installer option
  • More responsive "fuzzy" zoom
  • Option to become default browser
  • Opening downloads from download manager fixed
  • Password echoing

  • Improved panning/scrolling performance
  • Improved start-up time
  • Numerous bug fixes
  • Improved painting performance
  • Improved UI polish
  • Improved kinetic panning with directional locking
  • Ability to scroll frames
  • Improved zooming support

Installing Fennec

System requirements:

  • Recommended: Windows Mobile device running Windows Mobile Professional 6.0 of greater. 256Mb of RAM or greater, ARMv6 800MHz, 32Mb of free storage

  • Minimum: ARMv5, 128Mb RAM

  • While our goal is to run on a number of devices, our development and testing efforts have primarily focused on the HTC Touch Pro, Samsung Omnia II, and the Acer neoTouch devices.



If you're viewing this on an Windows Mobile Device, tap here to install Fennec 1.1 alpha 1 from the cab (or try the new faster .exe installer*)

If you don't have a Windows Mobile device, you can install the mobile version of Fennec on your desktop in order to test, provide feedback and build add-ons. The desktop versions are intended for development purposes only.


  • Windows
  • Mac OS X
  • Linux

* note: Since this alpha release of Fennec is unsigned, you will be prompted by the OS before loading each .dll and .exe in Fennec (20 separate prompts during installation and first run). We only recommend using the .exe installer on a device with the security turned off. If your device has security turned off, download the exe installer here



Known issues

  • Signing is targeted for Windows Mobile beta release

  • "Extract Error" that happens during installation bug 535370

  • Performance and stability can be degraded by an old profile bug 544917
  • Crash or freeze on HTC HD2 either on start up or early in run.

  • URL bar is slow to open
  • Rotating the display may cause painting or zooming errors
